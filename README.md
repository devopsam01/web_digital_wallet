# Digital Wallet Frontend

A React-based frontend application for a **Digital Wallet** platform.  
Supports user and admin authentication, wallet management, transactions, and audit logs.

---

## Features

- **User Authentication**
  - Login and registration for users and admins
  - Role-based navigation (user dashboard or admin dashboard)

- **Wallet Management**
  - Create, view, and manage wallets
  - Top-up and transfer funds
  - View transaction history

- **Admin Features**
  - Dashboard with metrics (total users, wallets, transactions, audit logs)
  - Charts showing system overview
  - Access to audit logs

- **Responsive UI**
  - Mobile-friendly and desktop-friendly design

---

## Tech Stack

- **Frontend:** React, Redux Toolkit, React Router
- **Charts:** Chart.js, react-chartjs-2
- **API Requests:** Axios
- **Styling:** CSS (modular for each page)
